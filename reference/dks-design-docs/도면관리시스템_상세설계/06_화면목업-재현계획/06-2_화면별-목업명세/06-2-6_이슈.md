# 목업: 이슈

## ① 목적
[[ACC]] Build의 **이슈(Issues)** 모듈을 동등 재현하는 목업 명세. 도면 위 **핀(pin)** 배치 → 우측 **상세폼** 입력 → **목록/필터** 조회 → **작성 모달** 생성의 4축을 픽셀 좌표 기반으로 정의. 1차 범위는 정적 가상 데이터 기반 UI 검증([[CLAUDE.md]] §4 Frontend-First).

## ② 배경/전제
- **편집 아님**: 핀/이슈는 [[DWG]] 원본을 건드리지 않고 2D 뷰어 위에 **오버레이**로 얹는다. 진실원천은 도면, 이슈는 그 위 주석 레이어.
- **좌표계**: 핀 위치는 뷰어의 시트 좌표(정규화 좌표 또는 시트 픽셀)에 고정. 줌/팬 시 동기 변환. → 변환 로직 상세는 [[06-2-2_시트]] 뷰어 절에 위임.
- **차별점(향후)**: 핀을 픽셀뿐 아니라 **설비 엔티티 ID**에도 바인딩 가능 — ACC에 없는 [[DKS]] 무기. Phase 2 [[TypeDB]] 온톨로지 연동 시 활성화. 목업 단계에서는 필드만 예약(❓).
- **공통 UI 재사용**: 앱셸·데이터테이블·우측 인스펙터·모달폼·빈상태·토스트는 공통 컴포넌트 위임 → [[06-1_공통UI-컴포넌트]].

## 대상 화면(다수)

### 화면 A — 뷰어 내 이슈 모드(핀 오버레이)
> 채울 것: [[06-2-2_시트]] 2D 뷰어 위에 이슈 핀 레이어를 띄운 상태. 핀 클릭 시 우측 인스펙터 열림.

#### 핀 시각 상태
> 채울 것: 핀 상태별 색/아이콘 — Open/In Review/Closed, 선택됨, 호버, 신규(미저장). 번호 배지 표기 규칙.

#### 핀 배치 인터랙션
> 채울 것: "이슈 추가" 토글 → 도면 클릭 → 임시 핀 생성 → 작성 모달 오픈 흐름. ESC 취소. 드래그 재배치 허용 여부 ❓.

#### 핀 ↔ 목록 연동
> 채울 것: 목록 행 호버/선택 시 해당 핀 하이라이트, 핀 클릭 시 목록 행 스크롤·선택 동기화.

### 화면 B — 이슈 목록 전용 화면(테이블)
> 채울 것: 뷰어 없이 프로젝트 전체 이슈를 데이터테이블로 보는 화면. 모듈 사이드바 진입점.

### 화면 C — 이슈 상세(우측 인스펙터 펼침)
> 채울 것: 핀/행 선택 시 우측 패널에 상세폼 전체 노출. 화면 A·B 공용 패널.

## 우측 상세폼 필드

### 헤더 영역
> 채울 것: 이슈 번호(#NNN), 제목(인라인 편집), 상태 드롭다운(Open/In Review/Closed), 닫기 버튼.

### 핵심 필드(ACC 동등)
> 채울 것: 아래 필드 표를 가상 데이터 스키마와 함께 확정.

| 필드 | 타입 | 비고 |
|---|---|---|
| 상태 | enum | Open / In Review / Closed |
| 유형(Type) | enum | 카테고리 TBD |
| 담당자(Assignee) | user | 단일 |
| 마감일(Due date) | date | |
| 위치(Location) | text/시트참조 | 시트명 + 핀 좌표 |
| 설명(Description) | textarea | |
| 우선순위 | enum | ❓ ACC 평가판 유무 확인 |

#### DKS 확장 필드(예약)
> 채울 것: `entityId`(연결 설비 엔티티), `discipline`(분야), `drawingTag`(도면 태그). Phase 0 파싱 산출물과 매핑. 목업에선 표시만, 편집 비활성 ❓.

### 첨부/마크업 연동
> 채울 것: 이슈에 마크업 스냅샷·사진·파일 첨부. [[06-2-3_마크업]] 핀↔마크업 연결, [[06-2-7_사진]] 위임.

### 활동/코멘트 타임라인
> 채울 것: 상태 변경 이력 + 코멘트 스레드. 가상 데이터 3~5건. 작성 입력창.

### 액션 바
> 채울 것: 저장/삭제/공유. 저장 시 토스트, 삭제 시 확인 모달.

## 목록/필터/모달

### 데이터테이블
> 채울 것: 컬럼(번호·제목·상태·담당자·마감일·위치). 정렬·컬럼토글·행 선택. 공통 데이터테이블 위임 → [[06-1_공통UI-컴포넌트]].

#### 컬럼/내보내기
> 채울 것: 기본 노출 컬럼 vs 토글 컬럼. CSV 내보내기 범위 ❓.

### 필터 패널
> 채울 것: 상태·담당자·유형·마감일 범위 필터. 다중 선택. 적용/초기화. 결과 카운트 표시.

#### 저장된 필터 / 빠른 필터
> 채울 것: "내 이슈" / "Open만" 등 프리셋. ACC 평가판 보유 여부 ❓.

### 작성 모달
> 채울 것: 핀 배치 직후 또는 "이슈 추가" 버튼으로 오픈하는 모달폼. 공통 모달폼 위임.

#### 필수/선택 필드 구성
> 채울 것: 제목·상태·위치(필수) / 담당자·마감일·설명(선택). 인라인 유효성.

#### 핀 좌표 캡처
> 채울 것: 모달 진입 시 클릭 좌표를 위치 필드에 프리필. 위치 없이 생성 허용 여부 ❓.

#### 생성 후 처리
> 채울 것: 저장 → 핀 확정(임시→정식) + 목록 갱신 + 인스펙터 자동 오픈 + 토스트.

### 빈 상태 / 로딩
> 채울 것: 이슈 0건 일러스트 + "이슈 추가" CTA. 스켈레톤 로딩.
