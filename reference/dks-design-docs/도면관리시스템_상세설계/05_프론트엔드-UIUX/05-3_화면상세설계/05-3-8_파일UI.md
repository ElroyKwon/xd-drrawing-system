# 파일 UI 화면설계

## ① 목적
- [[ACC Build]]의 "파일" 모듈(폴더형 [[CDE]])을 동등 재현하는 화면 상세설계.
- 소유 범위 = **폴더트리 + 파일테이블** 2분할 레이아웃. 파일/폴더 탐색·정렬·필터·진입의 단일 진실 정의처.
- 업로드·컨텍스트 메뉴·상세/버전 패널은 이 화면의 직접 부속 인터랙션으로 한정 포함.

## ② 배경/전제
- 핵심 스코프 제약: [[DWG]] 원본을 편집하지 않음. 파일 모듈은 **저장·분류·탐색·버전 보관소**(CDE)이며, 실제 렌더/마크업은 [[시트UI]]·뷰어 모듈이 담당.
- 파일 모듈은 폴더 계층의 메타 관리에 집중. 도면 진입 시 시트/뷰어로 위임(DRY).
- DKS 차별점: 파일 메타에 Phase 0 파싱 산출물(분야·도면번호·태그)을 컬럼/필터로 노출. 향후 "설비 엔티티 ID" 연결은 상세 패널에서 [[TypeDB]] 온톨로지로 바인딩(Phase 2, TBD).
- 1차 범위 = 2D 시트 도면 한정. 3D/Bridge 범위 밖.

## ③ 상세 목차

## 폴더 트리
좌측 고정 패널. CDE 폴더 계층을 트리로 표현하고 선택 시 우측 테이블 필터링.

### 트리 데이터 모델
- 노드 = {id, name, parentId, 권한, 자식 카운트}. 지연 로딩 여부 결정.
- 의도 메모: 폴더 깊이/카운트 표기 규칙, 루트 폴더 고정 항목(검토중/승인됨/도면 등) 정의.

### 트리 인터랙션
- 펼침/접힘, 선택 하이라이트, 드래그 이동(범위 내/외 TBD).
- 의도 메모: 단일선택 기준, 선택 상태와 URL 동기화 방식 채울 것.

#### 폴더 컨텍스트 액션
- 우클릭 → 새 폴더/이름변경/권한/삭제. (→ [[#업로드/컨텍스트 메뉴]]로 상세 위임)

#### 빈 상태 / 로딩
- 폴더 없음 빈상태, 스켈레톤 로딩 표기.

## 파일 테이블
우측 메인. 선택 폴더의 파일·하위폴더 목록. 공통 [[데이터테이블]] 컴포넌트 인스턴스.

### 컬럼 정의
- 기본: 이름(아이콘), 버전, 분야, 도면번호, 수정일, 수정자, 상태, 크기.
- 의도 메모: DKS 파싱 메타(분야·도면번호·태그) 컬럼을 ACC 기본 컬럼에 추가. 컬럼 토글 기본 노출/숨김 세트 확정 TBD.

### 정렬·필터·검색
- 컬럼 정렬, 분야/상태/태그 필터, 파일명 검색.
- 의도 메모: 필터 조합 규칙은 공통 데이터테이블에 위임, 이 문서는 파일 고유 필터(분야·도면번호)만 정의.

### 행 인터랙션
- 단일클릭 선택 → 상세 패널, 더블클릭 → 진입.
- 폴더 행 더블클릭 = 트리 하위 이동. 도면 파일 더블클릭 = [[시트UI]]/뷰어 진입.
- 의도 메모: 파일 타입별 진입 분기표(도면/PDF/이미지/기타) 채울 것.

#### 행 선택·다중선택
- 체크박스 다중선택 → 일괄 다운로드/이동/삭제 툴바.

#### 행 컨텍스트 메뉴
- 우클릭 액션 세트 (→ [[#업로드/컨텍스트 메뉴]]).

### 내보내기
- 목록 CSV/엑셀 내보내기. (공통 데이터테이블 기능 재사용)

## 업로드/컨텍스트 메뉴
파일 추가·관리 진입점. 드래그앤드롭 + 버튼 + 우클릭 메뉴 3경로.

### 업로드 플로우
- 드래그앤드롭 영역, 다중 파일 큐, 진행률, 성공/실패 토스트.
- 의도 메모: 업로드 직후 백엔드 [[DWG]] 파싱 트리거 → 진행상태 표시. 파싱 진행은 [[WebSocket]] 푸시 여부 TBD.

#### 업로드 대상 폴더·중복 처리
- 현재 폴더 기본 대상, 동명 파일 = 새 버전 vs 덮어쓰기 선택.
- 의도 메모: 버전 승계 규칙(→ [[#상세/버전 패널]]) 연계 정의.

#### 허용 파일 타입·용량
- DWG/PDF/이미지 허용, 제한 초과 처리. 구체 한도 TBD.

### 컨텍스트 메뉴 액션 정의
- 열기 / 다운로드 / 이름변경 / 이동 / 새 버전 업로드 / 권한 / 삭제.
- 의도 메모: 폴더 vs 파일 vs 다중선택별 액션 가용성 매트릭스 표로 채울 것.

#### 모달 폼
- 이름변경·이동·권한 = 공통 [[모달폼]] 재사용. 이 문서는 필드만 정의.

## 상세/버전 패널
우측 인스펙터. 행 선택 시 슬라이드인. 파일 메타 + 버전 이력 + 미리보기.

### 메타 정보 섹션
- 파일명, 분야, 도면번호, 태그, 크기, 생성/수정 정보, 상태.
- 의도 메모: DKS 파싱 메타 표기. **설비 엔티티 ID 연결 슬롯**(Phase 2 [[TypeDB]] 바인딩) 자리만 예약.

### 버전 이력 섹션
- 버전 리스트(V1·V2…), 업로드자·일시, 현재 버전 표시.
- 의도 메모: 버전 선택 → 미리보기 갱신. 두 버전 비교는 [[시트UI]] "시트 비교"로 위임(여기선 진입 버튼만).

#### 버전 액션
- 특정 버전 다운로드 / 현재로 설정 / 비교 진입.

### 미리보기 섹션
- 썸네일/[[PDF.js]] 경량 미리보기. 풀 렌더는 뷰어 위임.
- 의도 메모: 미리보기 소스(파싱 기하 JSON 썸네일 vs PDF 페이지) 결정 TBD.

### 연계 정보 섹션
- 해당 파일과 연결된 이슈/마크업 카운트 → 각 모듈 위임 링크.
- 의도 메모: Phase 2 — 엔티티 단위 영향도 진입점 예약(TBD).
