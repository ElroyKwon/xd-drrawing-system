# 2D 뷰어 셸 화면설계

## ① 목적
- 파싱된 [[DWG]] 기하 JSON을 웹 2D 뷰어로 렌더하는 **셸(레이아웃 골격)** 정의
- 소유 범위 = **뷰어 캔버스 + 툴바 도크 배치**만. 개별 도구 동작은 위임(아래 ⑤)
- [[ACC]] Build 시트 뷰어의 화면 골격을 동등 재현하되, 우리 흐름(뷰어+오버레이)에 맞춤

## ② 배경 / 전제
- **★스코프 제약**: DWG 원본 비편집. 흐름 = DWG → 백엔드 파싱 → 기하 JSON → 2D 렌더 → 오버레이(마크업/이슈/이벤트)
- 정밀 CAD 편집(osnap, 좌표 편집, [[SHX]] 정밀 폰트) **범위 밖**
- 1차 범위 = **2D 시트 한정**. 3D BIM 뷰어 범위 밖
- 렌더 기술 후보: [[PDF.js]](PDF 변환 경로) vs Canvas/[[WebGL]] 직접 렌더(기하 JSON) → ❓ TBD(아래)
- 이 셸은 **컨테이너**일 뿐 — 도구/패널 내부 로직은 각 소유 문서가 채움

---

## 캔버스 영역
> 채울 내용: 렌더 대상 영역의 좌표계·렌더 파이프라인·인터랙션(팬/줌)·오버레이 합성 순서

### 렌더 파이프라인
> 기하 JSON → 벡터 레이어 합성 → 화면. 렌더러 선택과 좌표 변환 책임 명시

#### 좌표계 / 단위
- 도면 월드좌표(mm) ↔ 뷰포트 픽셀좌표 변환 행렬 정의
- 줌 레벨 범위·기본 fit-to-screen 동작 명시. ❓ DPI 스케일 처리 방식 TBD

#### 레이어 합성 순서
- z-order: ① 베이스 도면(벡터) → ② 마크업 오버레이 → ③ 이슈 핀 → ④ 이벤트/하이라이트
- 각 레이어 on/off 토글은 [[#좌측 패널(마크업/이슈)]] 및 뷰 컨트롤과 연동

### 인터랙션
> 팬/줌/선택의 기본 제스처. 도구 활성 시 캔버스 입력 우선순위(hit-test) 규칙

#### 팬 / 줌
- 마우스 휠 줌(커서 기준), 스페이스+드래그 또는 중클릭 팬. 터치 핀치 ❓ TBD
- 줌 % 표시는 [[#하단 뷰 컨트롤/필름스트립]]과 동기화

#### 선택 / 히트테스트
- 클릭 시 hit-test 우선순위: 활성 도구 > 이슈 핀 > 마크업 > 엔티티
- **★차별점**: 클릭 대상이 파싱된 **설비 엔티티 ID**에 매핑되면 인스펙터에 엔티티 정보 노출(Phase 2 [[TypeDB]] 연동 훅). 상세는 [[05-3-6_이슈모듈]]로 위임

### 빈 상태 / 로딩
> 시트 미선택·파싱 진행중·렌더 실패 3종 상태 화면. 공통 빈상태 컴포넌트 재사용

## 우측 마크업 툴바
> 채울 내용: 캔버스 우측 세로 도크. 도구 버튼 그룹·활성 상태·확장 패널 배치만(도구 로직은 위임)

### 도크 레이아웃
> 세로 아이콘 바. 그룹 구분선·활성 하이라이트·툴팁 규격

#### 도구 그룹 구성
- 선택 / 펜 / 도형(사각·원·선) / 화살표 / 텍스트 / 측정(❓ 범위 포함 여부 TBD)
- 각 버튼 = 아이콘 + 단축키 툴팁. **도구별 상세 동작은 [[05-3-5_마크업툴]]로 위임**

#### 도구 옵션 팝오버
- 도구 선택 시 색상·두께·폰트 옵션 팝오버 노출 위치(도크 좌측 인접) 정의
- 옵션 상태 영속화 ❓ TBD(세션/사용자 프리퍼런스)

### 활성/비활성 규칙
> 뷰 전용 모드 vs 마크업 모드 전환 시 툴바 활성화 조건. 권한(읽기전용 사용자) 연동

## 하단 뷰 컨트롤/필름스트립
> 채울 내용: 캔버스 하단 가로 바. 줌 컨트롤·페이지 네비·다중 시트 썸네일 스트립

### 뷰 컨트롤 바
> 줌 %, fit, 회전(❓), 레이어 토글 등 빠른 제어. 캔버스 상태와 양방향 바인딩

#### 줌 / 핏 컨트롤
- 줌 -, %, +, fit-to-screen, 1:1 버튼. [[#캔버스 영역]] 변환 행렬과 동기화

#### 레이어 토글 단축
- 마크업/이슈/이벤트 레이어 빠른 on/off(좌측 패널과 중복 진입점, 단일 상태 SoT)

### 필름스트립(다중 시트)
> 한 도면집 내 여러 시트 썸네일 가로 스크롤. 현재 시트 하이라이트·클릭 전환
- 썸네일 출처(파싱 산출 미리보기) ❓ TBD. 대량 시트 가상 스크롤 적용 여부 TBD

## 좌측 패널(마크업/이슈)
> 채울 내용: 캔버스 좌측 슬라이드 패널. 현재 시트의 마크업·이슈 목록 + 캔버스 연동

### 패널 셸 / 탭 구성
> 마크업 탭 / 이슈 탭 토글. 접기-펼치기, 폭 드래그 리사이즈

#### 마크업 목록 탭
- 현재 시트 마크업 리스트(작성자·시각·색). 항목 클릭 → 캔버스 해당 위치 포커스
- 목록 자체 CRUD/필터는 [[05-3-5_마크업툴]] 소유, 여기선 **컨테이너 배치만**

#### 이슈 목록 탭
- 현재 시트 이슈 핀 리스트 + 상태 배지. 클릭 → 핀 포커스 + 우측 인스펙터 오픈
- **★엔티티 바인딩**: 이슈가 설비 엔티티 ID에 묶인 경우 엔티티 칩 표시(훅). 상세 [[05-3-6_이슈모듈]] 위임

### 캔버스 연동 규칙
> 목록 ↔ 핀/마크업 양방향 하이라이트. 패널 열림이 캔버스 가용폭에 미치는 영향(리플로우)

### 우측 인스펙터 관계
> 좌측=목록 진입, 우측=선택 항목 상세. 공통 인스펙터 패널 컴포넌트 재사용 명시(중복 정의 금지)
