# 시트 비교 UI 화면설계

## ① 목적
- 동일 도면의 두 버전(또는 두 시트)을 겹쳐 보고 **변경 영역(diff)을 시각적으로 식별**하는 화면 설계.
- [[ACC]] Build "시트 비교" 기능 동등 재현. 비교 진입(다이얼로그/시트피커) ~ 결과 오버레이 뷰까지를 소유.

## ② 배경 / 전제
- ★스코프 제약: [[DWG]] 원본 미편집. `DWG → 백엔드 파싱 → 기하 JSON → 웹 2D 뷰어 렌더 → 오버레이`. 비교는 **렌더 결과(래스터/벡터) 위 오버레이 연산**이며 CAD 지오메트리 재계산이 아님.
- 비교 대상 = **2D 시트 한정** (3D BIM 비교 범위 밖).
- 뷰어 렌더 엔진·좌표계·줌/팬은 [[05-3-2_시트목록·2D뷰어UI]]가 소유 → 본 문서는 그 뷰어 위 "비교 모드"만 다룸(DRY).
- 버전 데이터 출처 = [[CDE]] 파일 버전 이력. 마크업/이슈 핀과의 결합은 향후 [[TypeDB]] 엔티티 ID 바인딩으로 확장(Phase 2).

---

## 비교 다이얼로그 / 시트피커
> 의도: 비교 시작 진입점. 어떤 두 대상을 고를지(버전 vs 버전, 시트 vs 시트) UX 정의.

### 진입 동선
> 의도: 뷰어 툴바 "비교" 버튼 / 시트목록 우클릭 / 버전 이력 패널에서의 진입 경로 3종 기술.

### 비교 모드 선택
> 의도: ⓐ 같은 시트의 버전 비교 ⓑ 서로 다른 시트 비교 — 탭/라디오 전환.

#### 기준(Base) 시트 선택
> 의도: 좌측 슬롯. 기본값 = 현재 열린 시트의 최신 버전. 썸네일 + 도면번호 + 버전 라벨.

#### 대상(Compare) 시트 선택
> 의도: 우측 슬롯. 버전 드롭다운(이력 리스트) 또는 다른 시트 검색 피커.

### 시트피커 컴포넌트
> 의도: 검색·필터(분야/도면번호/태그) 가능한 목록. 공통 [[데이터테이블]] 패턴 재사용 → 상세는 [[05-2_공통UI컴포넌트]] 위임.

#### 분야·도면번호·태그 필터
> 의도: Phase 0 파싱 산출물(분야/도면번호/태그) 기반 필터. 차별점 — ACC엔 없는 메타 필터.

### 다이얼로그 액션
> 의도: [비교 실행] / [취소]. 유효성(동일 대상 선택 방지, 미선택 비활성) 처리.

### 빈 상태 / 에러
> 의도: 비교 가능한 버전이 1개뿐일 때, 렌더 실패 시 빈상태·토스트. 공통 패턴 위임.

## 오버레이 결과 뷰
> 의도: 두 렌더를 겹쳐 diff를 보여주는 메인 결과 화면. 시각 표현 방식 정의.

### 레이아웃 모드
> 의도: 결과 뷰의 배치 옵션. ACC 벤치마크 기준 모드 열거.

#### 오버레이(겹침) 모드
> 의도: Base/Compare를 한 캔버스에 중첩. 색상 구분(추가=초록·삭제=빨강·동일=회색 등) — 정확한 색값 TBD.

#### 나란히(Side-by-Side) 모드
> 의도: 좌우 2분할. 줌/팬 동기화 옵션.

#### 스와이프(Swipe) 모드
> 의도: 슬라이더 핸들로 두 버전을 가르는 와이프 비교.

### diff 시각화 방식
> 의도: 색상 합성(blend) vs 윤곽선 강조 vs 변경영역 하이라이트 박스 중 채택 방식 결정.
> ❓ TBD: 픽셀 diff(래스터 합성) vs 벡터 요소 diff 중 1차 구현 범위.

### 변경 영역 네비게이션
> 의도: 감지된 변경 영역 간 [이전]/[다음] 이동, 변경 영역 줌-투-핏.

### 뷰어 인터랙션 연동
> 의도: 줌/팬/회전은 [[05-3-2_시트목록·2D뷰어UI]] 뷰어 컨트롤 재사용. 비교 모드 고유 단축키만 여기 정의.

### 오버레이 위 주석 표시 정책
> 의도: 비교 모드에서 기존 [[마크업]]/이슈 핀을 표시할지 여부·토글. 상세 동작은 [[05-3-3_마크업UI]] 위임.

## 비교 문서 패널 / 가시성 토글
> 의도: 결과 뷰 우측 인스펙터. 두 비교 대상의 메타와 레이어 on/off 제어.

### 인스펙터 패널 구조
> 의도: 공통 우측 인스펙터 셸 재사용 → [[05-2_공통UI컴포넌트]]. 본 절은 비교 전용 콘텐츠만.

#### 비교 대상 메타 카드
> 의도: Base/Compare 각각 도면번호·버전·수정일·작성자 카드. 출처 = [[CDE]] 버전 이력.

#### 색상 범례(Legend)
> 의도: diff 색상 의미(추가/삭제/동일) 범례. 색값은 diff 시각화 결정 후 확정.

### 가시성 토글
> 의도: 레이어별 표시/숨김 스위치.

#### Base / Compare 레이어 토글
> 의도: 두 도면 레이어 개별 on/off, 불투명도 슬라이더(0~100%).

#### 변경영역 하이라이트 토글
> 의도: diff 하이라이트만 별도 on/off.

#### 주석 레이어 토글
> 의도: 마크업/이슈 핀 오버레이 on/off (표시 정책 절과 연동).

### 결과 내보내기 / 공유
> 의도: 비교 결과 [[PDF]]/이미지 스냅샷 내보내기, 이슈로 전환(변경영역→이슈 생성) 진입점.
