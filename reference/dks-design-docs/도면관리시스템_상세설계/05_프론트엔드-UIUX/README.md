# 프론트엔드 / UIUX 개요

## 목적

- 청주 사업장 납품 [[도면관리 시스템]]의 **프론트엔드(FE) 설계 문서 진입점**.
- 이 README는 FE/UIUX 하위 문서의 **인덱스(허브)** 역할만 수행 — 상세 화면/컴포넌트 사양은 각 하위 문서가 소유(DRY).
- 1차 목표: [[ACC]](Autodesk Construction Cloud) Build 제품의 화면/상호작용을 웹에서 **동등 재현**.

## 배경 / 전제

- ★스코프 제약(모든 FE 문서의 대전제):
  - [[DWG]] 원본을 **편집하지 않음**. 흐름 = DWG → 백엔드 파싱 → 기하 JSON → 웹 2D 뷰어 렌더 → 그 위에 마크업/이슈/이벤트 **오버레이**.
  - FE는 **"뷰어 + 주석/이벤트 오버레이"** UI — CAD 편집기가 아님.
  - 정밀 CAD 편집(osnap 스냅, 좌표 편집, [[SHX]] 폰트 정밀 렌더)은 범위 밖. (과거 Development_Design 초안의 CAD-editor 가정 폐기.)
  - 1차 범위 = **2D 시트(도면) 한정**. 3D BIM 뷰어/Bridge 범위 밖.
- 벤치마크: ACC Build 37화면 분석 → [[_ACC-Build-화면분석-재현설계]] 참조(원천 사양).
- 차별점(향후): 핀/마크업/이슈를 픽셀뿐 아니라 **설비 엔티티 ID**에도 바인딩 → Phase 2 [[TypeDB]] 온톨로지 연동([[AI채팅]]·영향도 검색). FE는 이 바인딩을 담을 데이터 슬롯을 선반영.

## 화면 설계 인덱스

> ACC 37화면 분석 기준 모듈별 화면 카탈로그. 각 항목은 `화면별-사양/`의 상세 문서로 드릴다운(작성 예정).

#### 모듈 화면 목록

- **홈** — 위젯 대시보드(위젯카드 배치·요약 지표). → 상세 ❓TBD.
- **시트** — 도면 목록(데이터테이블) + 2D 뷰어. → [[2D뷰어-렌더링]] 위임 예정.
- **마크업** — 펜·도형·화살표 그리기 오버레이. → [[오버레이-마크업-이슈]] 위임 예정.
- **이슈** — 핀 배치 + 상세 폼(모달) + 목록 + 필터. → 위임 예정.
- **시트 비교** — 버전 오버레이 diff(2D). → 위임 예정.
- **파일** — 폴더형 [[CDE]](공통 데이터 환경). → 위임 예정.
- **사진** — 현장 사진 갤러리/메타. → 위임 예정.

#### 공통 UI 패턴 (전 모듈 공유)

- 앱셸: 모듈 사이드바 + 프로젝트 스위처 → [[앱셸-네비게이션]] 소유.
- 데이터테이블: 정렬·컬럼 토글·필터·내보내기 → [[공통-UI-컴포넌트]] 소유.
- 우측 인스펙터 패널 / 위젯카드 / 모달 폼 / 빈 상태 / 토스트 → [[공통-UI-컴포넌트]] 소유.
