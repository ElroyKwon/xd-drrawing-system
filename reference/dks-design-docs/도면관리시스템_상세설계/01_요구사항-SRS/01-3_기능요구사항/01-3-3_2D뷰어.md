# 2D 도면 뷰어 기능요구

## ① 목적
- 청주 사업장 [[도면관리 시스템]]의 핵심 화면인 **2D 시트 뷰어**의 뷰 조작 기능요구 정의.
- 소유 범위 한정: **줌 / 팬 / 맞춤 / 레이어 / 시트전환**. (그 외는 위임)
- [[ACC Build]] 시트 뷰어를 동등 재현하되, "뷰어 + 오버레이" 원칙 준수.

## ② 배경 / 전제
- **핵심 스코프 제약**: [[DWG]] 원본을 편집하지 않음. 흐름 = DWG → 백엔드 파싱 → 기하 JSON → 웹 2D 렌더 → 마크업/이슈/이벤트 오버레이.
- 본 뷰어는 **CAD 편집기가 아님**: osnap 스냅, 좌표 편집, [[SHX]] 폰트 정밀 렌더는 범위 밖.
- 1차 범위 = **2D 시트 한정**. 3D [[BIM]] 뷰어/Bridge 제외.
- 렌더 대상 = 백엔드 산출 **기하 JSON**(벡터) + 폴백 [[PDF.js]] 래스터/벡터. ❓최종 렌더 파이프라인 확정 필요(아래 TBD).
- 차별점 메모: 핀/마크업/이슈를 픽셀뿐 아니라 **설비 엔티티 ID**에도 바인딩 → 향후 [[TypeDB]] 온톨로지 영향도/AI채팅 연동의 진입점. (상세는 [[01-3-5_이슈]]·온톨로지 트랙에 위임)

## ③ 상세 목차 (기능요구 골격)

### 뷰 조작(줌/팬/맞춤)
> 채울 것: 마우스/터치/키보드 입력별 뷰 변환 동작과 한계값 정의.

#### 줌(Zoom)
- 휠 줌 (커서 기준 줌-투-포인트), 줌 인/아웃 버튼, 핀치 줌.
- 최소/최대 배율 한계 ❓(TBD: 예 25%~3200%, ACC 기준 확인).
- 줌 단계 비율, 줌 시 마크업 핀 크기 고정(스케일 불변) 여부.

#### 팬(Pan)
- 드래그 팬(스페이스+드래그 / 휠클릭 / 빈영역 드래그 중 택), 관성 스크롤 여부 ❓.
- 패닝 경계(시트 영역 밖 과도한 이동 제한) 정책.

#### 맞춤(Fit / Zoom-to)
- 화면맞춤(Fit to screen), 너비맞춤, 실제크기(1:1), 선택영역 줌(rubber-band zoom).
- 초기 진입 시 기본 = 화면맞춤. 시트 전환 시 뷰 상태 유지 vs 재맞춤 ❓.

#### 회전 / 미니맵 (선택)
- 90° 회전 지원 여부 ❓(ACC 미지원이면 제외).
- 미니맵/오버뷰 네비게이터 표시 여부.

### 시트 전환(필름스트립)
> 채울 것: 시트 간 빠른 이동 UX와 상태 유지 규칙.

#### 필름스트립 썸네일 바
- 하단/측면 썸네일 스트립, 현재 시트 하이라이트, 클릭 전환.
- 썸네일 생성 출처(백엔드 미리보기 이미지) ❓.

#### 이전/다음 네비게이션
- 키보드(←/→, PgUp/PgDn), 시트 드롭다운 점프, 도면번호 검색 점프.
- 시트 목록(좌측/별도 화면)과의 연동 → 상세 [[01-3-2_시트목록]] 위임.

#### 전환 시 상태 처리
- 줌/팬 상태 유지 정책, 활성 마크업 도구 유지 여부, 미저장 마크업 경고.

### 레이어/표시 설정
> 채울 것: 도면 레이어 토글과 오버레이 표시 옵션. (편집 아님, 표시만)

#### CAD 레이어 토글
- 파싱된 DWG 레이어 목록 표시 → on/off 토글(읽기 전용).
- 레이어 색상/선두께는 원본 기하 JSON 값 사용. 레이어 편집 불가.
- 레이어 수 과다 시 그룹핑/검색 ❓.

#### 오버레이 표시 옵션
- 마크업 표시/숨김, 이슈 핀 표시/숨김, 이벤트 핀 표시/숨김(레이어별 독립 토글).
- 표시 설정 영속화(사용자별 세션 저장) 여부 ❓.

#### 배경/렌더 옵션
- 흑/백 배경 반전, 라인 안티앨리어싱, 측정 단위 표시(있을 경우).

### 좌측 패널(마크업/이슈)
> 채울 것: 뷰어 내 좌측 패널의 "뷰 연동" 책임만 정의. 마크업/이슈 본기능은 위임.

#### 패널 구조 / 탭
- 좌측 패널 = 마크업 목록 탭 + 이슈 목록 탭(+ 표시설정). 접기/펼치기.

#### 항목 ↔ 뷰 포커스 연동
- 목록 항목 클릭 → 해당 핀/마크업으로 **줌·팬 포커스 이동**(zoom-to-pin).
- 뷰에서 핀 선택 → 패널 항목 하이라이트(양방향 동기화).

#### 위임 경계 (DRY)
- 마크업 생성/편집 도구(펜·도형·화살표) → [[01-3-4_마크업]].
- 이슈 핀 생성/상세폼/필터 → [[01-3-5_이슈]].
- 본 문서는 **뷰 포커스·표시 토글**만 소유.

### 성능 목표
> 채울 것: 측정 가능한 NFR 후보. 과장 금지, 미확정은 TBD.

#### 렌더 성능
- 초기 시트 로드 시간 목표 ❓(TBD: 측정 후 확정).
- 줌/팬 인터랙션 프레임율 목표(체감 60fps 지향) ❓.
- 대용량 도면(엔티티 수 많은 시트) 처리 한계 ❓(가상화/타일링 필요성 검토).

#### 데이터 로딩
- 기하 JSON 점진적 로드/캐싱 전략, 썸네일 선로딩.
- 마크업/이슈 오버레이 실시간 갱신 채널 = [[WebSocket]] (다중 사용자 동시 열람 시) ❓.
