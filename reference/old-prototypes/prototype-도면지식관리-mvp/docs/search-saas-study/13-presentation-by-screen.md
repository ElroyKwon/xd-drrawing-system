## A. 머리말 — 이 목업의 목적

**무엇이 아닌가**
- 상용 서비스 코드가 아니다. 백엔드·DB·권한·업로드·PWA 오프라인은 **의도적으로 미구현**이다.
- 성능 벤치마크·부하 테스트 대상이 아니다.

**무엇인가**
- 이전 `D:\_Project\prototype-도면지식관리\`에서 8인 페르소나 검증 결과 **자발적 지불 의사 0명 / 평균 채택 2.3/10**이었다는 통합 결론(`05-통합분석-결론.md`)의 재시도다.
- 그 실패의 원인이 "기술·기능"이 아니라 "**서사·포지셔닝·사용 문맥**"에 있다는 가설(`07-actual-code-verification.md §핵심 메시지`)을 **극도로 좁은 Layer 0 MVP + 14개 세밀한 UX 원칙**으로 다시 증명 시도.
- 이 목업을 **시연**하고, **페르소나 재인터뷰**하고, 지불 의사·채택의사 변화를 측정한 뒤 go/no-go를 결정하는 것이 다음 단계.

**프레젠테이션 시 권장 안내 문구**
> *"이것은 상용 코드가 아닙니다. 여러분과 함께 '이것이 우리가 만들고자 하는 서비스의 모습이 맞는가'를 논의하기 위한 시연 프로토타입입니다. 좋아 보이는 것·이상한 것·빠진 것을 모두 말씀해 주세요."*

### D-2. 각 위젯 세부

1. **최근 본 도면** — `useRecentDocs` 훅, LRU 20 중 상위 5건, DocCard dense 레이아웃. hydrate 전 스켈레톤.
2. **즐겨찾기** — `useFavorites` 훅, unique doc_id 5건. 페이지 북마크는 별도 화면에서 보여줌.
3. **최근 업데이트** — `documents.sort(last_updated DESC)` 상위 5건. suffix에 수정일 노출.
4. **학제별 도면 수** — 4분야(전기·기계·소방·시설) 막대 차트. 색상은 `disciplineBarStyles`로 고정. 도면함 그리드 뷰 링크.
5. **설비 태그 빠른 진입** — `docEntityLinks`에서 태그별 링크 수 상위 10종을 태그 핀 칩으로. 클릭 → `/foundation/equipment`.
6. **Foundation ScoreCard** — Layer A(기본 검색) 11요건 + Layer C(도면 특화) 7요건, 18행 표. 각 행 PASS / 약함 표시. 07번 문서 검증 근거.

### D-4. 시나리오

### E-2. 기능

- **URL querystring 기반 상태 동기** — `q`, `discipline`, `location`, `tag`, `sort`, `view`, `sel` 7개. 즐겨찾기/공유 가능.
- **Fuse.js 검색** — `searchDocuments(q)` 호출, 제목·타입·스니펫·drawing_number 다축 색인.
- **정렬 3종** — 관련도순(디폴트) / 최근수정순 / 도면번호순.
- **리스트 ↔ 그리드** — 토글 (StarNote DNA #6). 그리드 모드에서는 우측 미리보기 숨김.
- **활성 필터 칩** — 각 필터를 한 번에 지울 수 있는 `×` 칩 (q · discipline · location · tag).
- **자동 선택** — 결과 있으면 첫 번째 도면 자동 선택. 필터 변경 시 `sel` 재검증.
- **T4 썸네일 지연 로딩** — `IntersectionObserver`로 뷰포트 진입 시에만 PDF 첫 페이지 렌더. 썸네일은 `PdfThumbnail` 컴포넌트가 캐시.

## Q. 현재 한계 (투명 공개)

### Q-1. 기술·시스템
- **백엔드 없음** — 모두 정적 JSON + localStorage. 다중 사용자·실시간 협업·감사 추적 불가
- **인증·권한 없음** — 누구나 모든 데이터에 접근. 업무용 서비스엔 치명적
- **PWA 오프라인 없음** — 현장 Wi-Fi 음영 대응 불가 (조사 1순위 미구현)
- **업로드 없음** — 새 도면 추가 불가
- **검색 엔진 클라이언트 Fuse.js** — 수백~수천 건까지만. 엔터프라이즈 스케일엔 서버 검색 필요

### Q-2. 데이터
- `dwg-*.pdf` 대부분 **0 byte 더미** — TextLayer 활성해도 글자 없음
