## 질문지

### Q1. 경험 평가 (리커트 5점)
1. 기존 방식 대비 **얼마나 빠르게** 도면을 찾았나
2. 답을 **얼마나 신뢰할 수 있었나**
3. **매뉴얼 없이** 바로 사용 가능했나
4. **모바일**에서 사용 경험
5. **매일 쓸 것 같은가**

### Q2. 정성
1. 오늘 시연 중 **가장 인상적인 한 기능**
2. **없으면 안 되는 기능 TOP 3**
3. **빠져서 아쉬운 기능**
4. 기존 NAS·엑셀에서 **이 서비스로 바꾸려면 무엇** 필요
5. 주변 동료에게 **어떻게 소개**하시겠나

### Q3. AI 인사이트 (서비스 2)
1. 답을 **신뢰**할 수 있었나
2. **근거** 확인이 쉬웠나
3. 답이 **30초 안에** 나왔나
4. **이 답을 그대로 믿고 행동**할 수 있겠나
5. ChatGPT와 비교하면

### Q4. 구매·가격
1. 월 300만 / 500만 / 1000만 원이면 어떤 반응
2. 사이트당 일시불 3천만 / 1억 / 3억이면
3. **결재 권한**은 누구이고, **설득 증거**로 뭐가 필요한가
4. 알람·보고서·브리핑 **어느 모듈**에 가장 지불 의사
5. 보고서 **한 건당 가치 가격**

### Q5. 위험·우려
1. AI가 **틀리면** 어떤 대가
2. **보안**상 우려되는 데이터
3. **운영 팀 문화**에서 받아들이기 힘든 점

### Q6. 갭 탐지
1. 이 서비스 **없었을 때** 어떻게 하고 있었나
2. 이 서비스 **있다면 안 할 일**
3. 이 서비스 **대신할 수 있는 것**
