✅ 긴급 분석이 필요한 경우 - ✅ 신규 배터리 초기 분석 (프로파일 생성 전) - ✅ 배터리 모델을 알 수 없는 경우 - ✅ 임시 운영 (1~2일) #### 사용하지 말아야 할 경우: - ❌ 장기 운영 (1주 이상) - ❌ 정밀 분석이 필요한 경우 - ❌ 배터리 모델을 알고 있는 경우 - ❌ 중요 안전 결정 시 ### 특징 ```yaml # GENERIC_ESS_SAFE.yaml thresholds: thermal_safety: critical: 1.0 # °C/min (가장 엄격 - NMC보다 낮음) warning: 0.5 # 매우 보수적 spc_balance: critical: 2.0 # σ (엄격) warning: 1.0 physical_limits: voltage: operating_min: 2.5 # 넓은 범위 (LFP~NMC 포괄) operating_max: 4.3 ``` ### ⚠️ 중요 경고 - **오탐(False Positive) 가능성 높음**: 임계값이 매우 엄격하므로 정상 상황도 경고로 표시될 수 있음 - **정확도 낮음**: 배터리 타입별 특성을 고려하지 않음 - **임시 사용만**: 가능한 빨리 정확한 프로파일로 변경 - **전문가 검토 필수**: 분석 결과를 반드시 검토 ### 다음 단계 GENERIC_ESS_SAFE 사용 후: 1. 배터리 정보 수집: - 제조사 (LG, CATL, Samsung 등) - 화학 구조 (LFP, NMC, NCMA) - 모델명 및 사양서 2. 정확한 프로파일 선택: - 기존 프로파일 사용 - 또는 새 프로파일 생성 3. Phase 2 재실행: - 정확한 임계값으로 재분석 - 정확도 3배 이상 향상 --- ## ⚠️ 주의사항 ### 프로파일 잘못 선택 시 문제점 #### 1. LFP를 NMC로 잘못 인식 ``` LFP 실제 전압: 3.6V (만충전 정상) ↓ NMC 프로파일 적용 (4.2V 기준) ↓ ❌ 저전압 경고 발생 (오탐) ❌ 과방전으로 오인 ``` #### 2. NMC를 LFP로 잘못 인식 ``` NMC 실제 전압: 4.1V (정상) ↓ LFP 프로파일 적용 (3.65V 기준) ↓ ❌ 과충전 경고 발생 (오탐) ❌ 위험 상황으로 오인 ``` #### 3. dT/dt 임계값 차이 ``` 실제 온도 상승: 1.8°C/min ↓ LFP 프로파일 (임계값 2.0°C/min) → ✅ NORMAL (정상) NMC 프로파일 (임계값 1.5°C/min) → ⚠️ CRITICAL (위험) ``` **→ 프로파일 선택이 매우 중요합니다!** --- ## 📚 참고 자료 ### 배터리 화학 구조 - **LFP (LiFePO₄):** 인산철 양극재 → 안정적, 저전압 - **NMC (LiNiMnCoO₂):** 니켈/망간/코발트 산화물 → 고에너지, 민감 - **NCA (LiNiCoAlO₂):** 니켈/코발트/알루미늄 산화물 → 최고 에너지, 매우 민감 ### 제조사별 특성 - **CATL (Contemporary Amperex Technology):** 세계 1위, LFP 전문 - **Samsung SDI:** 한국, NMC 전문 - **LG Energy Solution:** 한국, NMC/NCMA - **Tesla/Panasonic:** NCA (Model S/X), LFP (Model 3 Standard) --- ## ✅ 체크리스트 ### 프로파일 선택 전 확인 - [ ] 배터리 제조사 확인 (LG?
