## 파일 관리 규칙

### 1. 경로 변경 사항 (2025-12-26)
```diff
- api_data/YYYY-MM-DD/채널명.csv
+ data/source/api/YYYY-MM-DD/채널명.csv

- uploads/파일명.csv
+ data/source/upload/YYYYMMDD_HHMMSS/파일명.csv

- data/*_tags.txt
+ data/tags/*_tags.txt

- results/*/phase1, phase2, phase3, final
+ results/*/phase1_convert, phase2_validate, phase3_insights, phase4_report

- config/battery_profiles/*.yaml
+ config/battery_profiles/profiles/*.yaml
```

### 2. 파일 명명 규칙
- **RUN_ID 형식**: `YYYYMMDD_HHMMSS_배터리타입`
  - 예: `20251213_153045_NMC_LG`
- **로그 파일**: `{모드}_{YYYY-MM-DD}.log`
  - 예: `historical_2025-12-24.log`

### 3. 백업 및 복구
- 마이그레이션 시 자동 백업: `_backup_YYYYMMDD_HHMMSS/`
- 백업에서 복구 가능

### 4. 삭제 금지 항목
- `data/source/` 전체
- `results/*/metadata.json`
- `config/state/` 전체
