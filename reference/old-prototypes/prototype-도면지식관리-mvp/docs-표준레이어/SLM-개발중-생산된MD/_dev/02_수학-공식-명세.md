## 📋 개요

### 목적
배터리 표준 파일에서 4대 핵심 알고리즘을 실행하여 위험 지표를 계산.

### 4대 핵심 알고리즘 (설계안)

| 알고리즘 | 목적 | 구현 상태 |
|---------|------|----------|
| **1. 열적 안전성 (Thermal Safety)** | dT/dt 계산 | ✅ 구현 완료 |
| **2. 셀 밸런싱 (SPC)** | Z-Score 계산 | ✅ 구현 완료 |
| **3. 내부 단락 (ISC)** | 전압/온도/전류 종합 분석 | ⚠️ 향후 구현 예정 |
| **4. 데이터 무결성 (Integrity)** | 물리 법칙 검증 | ✅ 구현 완료 (일부) |

## 🔥 Algorithm 1: 열적 안전성 (Thermal Safety) - ✅ 구현 완료

### 수식

```
dT/dt = (T_current - T_previous) / Δt

where:
  T_current  = 현재 시점 평균 온도 (t_avg)
  T_previous = 이전 시점 평균 온도 (t_avg)
  Δt         = 시간 간격 (분 단위)
```

**⚠️ 주의**:
- **설계안**: t_max (최대 온도) 사용
- **실제 구현**: t_avg (평균 온도) 사용
- **이유**: t_avg가 전체 셀의 온도 분포를 더 잘 반영하여 안정적

### 임계값 (배터리 프로파일별로 다름)

| 등급 | dT/dt 범위 | 위험도 | 조치 |
|------|-----------|--------|------|
| **NORMAL** | < warning | 정상 | 모니터링 |
| **WARNING** | warning ~ critical | 주의 | 관찰 강화 |
| **CRITICAL** | > critical | 위험 | 즉시 조치 |

**프로파일별 임계값 예시**:
- **NMC_LG**: critical=1.5°C/min, warning=1.0°C/min
- **LFP_CATL**: critical=2.0°C/min, warning=1.5°C/min
- **GENERIC_ESS_SAFE**: critical=1.0°C/min (가장 엄격)

### Python 구현

**실제 구현 위치**: `scripts/phase2_validate_safety.py` (라인 79-105)

```python
# scripts/phase2_validate_safety.py
import polars as pl

def calculate_thermal_safety(df: pl.DataFrame, threshold_critical: float, threshold_warning: float) -> pl.DataFrame:
    """열적 안전성 계산 (dT/dt)

    Args:
        df: 배터리 표준 파일 (Parquet)
           필수 컬럼: timestamp, rack_id, t_max

    Returns:
        컬럼 추가된 DataFrame
        - dtdt: 온도 변화율 (°C/min)
        - thermal_risk: NORMAL/WARNING/CRITICAL
    """

    result = df.sort(['rack_id', 'timestamp']).with_columns([
        # 1. 이전 시점 온도 (Rack별)
        pl.col('t_max').shift(1).over('rack_id').alias('t_prev'),

        # 2. 시간 간격 (분 단위)
        (
            pl.col('timestamp').diff().over('rack_id').dt.total_seconds() / 60.0
        ).alias('delta_t_min'),
    ]).with_columns([
        # 3. dT/dt 계산
        (
            (pl.col('t_max') - pl.col('t_prev')) / pl.col('delta_t_min')
        ).alias('dtdt')
    ]).with_columns([
        # 4. 위험도 분류
        pl.when(pl.col('dtdt') > 2.0)
          .then(pl.lit('CRITICAL'))
          .when(pl.col('dtdt') > 1.0)
          .then(pl.lit('WARNING'))
          .otherwise(pl.lit('NORMAL'))
          .alias('thermal_risk')
    ])

    return result


# 물리 검증
def validate_thermal(dtdt: float) -> bool:
    """물리적으로 가능한 온도 변화율인지 검증

    Returns:
        True: 정상, False: 센서 오류 의심
    """
    # 배터리가 100°C/min 이상 변화는 물리적으로 불가능
    return -100.0 <= dtdt <= 100.0
```

## 📊 Algorithm 2: 셀 밸런싱 (SPC - Statistical Process Control)

### 수식

```
Z-Score = (X - μ) / σ

where:
  X = 개별 셀 전압 (v_avg)
  μ = 평균 전압 (전체 Rack 기준)
  σ = 표준편차
```

### 임계값

| 등급 | Z-Score 범위 | 해석 | 조치 |
|------|-------------|------|------|
| **NORMAL** | \|Z\| < 2.0σ | 정상 범위 | 모니터링 |
| **WARNING** | 2.0σ ≤ \|Z\| < 3.0σ | 주의 범위 | 밸런싱 점검 |
| **CRITICAL** | \|Z\| ≥ 3.0σ | 이상 범위 | 즉시 점검 |

### Python 구현 (Global - 수정 불가)

```python
# config/global/formulas/spc_balance.py
import polars as pl

def calculate_spc_balance(df: pl.DataFrame) -> pl.DataFrame:
    """셀 밸런싱 분석 (Z-Score)

    Args:
        df: 배터리 표준 파일
           필수 컬럼: timestamp, rack_id, v_avg

    Returns:
        컬럼 추가된 DataFrame
        - v_mean: 전체 평균 전압
        - v_std_dev: 표준편차
        - z_score: Z-Score 값
        - spc_risk: NORMAL/WARNING/CRITICAL
    """

    result = df.with_columns([
        # 1. 전체 평균 및 표준편차 (시간대별)
        pl.col('v_avg').mean().over('timestamp').alias('v_mean'),
        pl.col('v_avg').std().over('timestamp').alias('v_std_dev'),
    ]).with_columns([
        # 2. Z-Score 계산
        (
            (pl.col('v_avg') - pl.col('v_mean')) / pl.col('v_std_dev')
        ).alias('z_score')
    ]).with_columns([
        # 3. 위험도 분류
        pl.when(pl.col('z_score').abs() >= 3.0)
          .then(pl.lit('CRITICAL'))
          .when(pl.col('z_score').abs() >= 2.0)
          .then(pl.lit('WARNING'))
          .otherwise(pl.lit('NORMAL'))
          .alias('spc_risk')
    ])

    return result


# 물리 검증
def validate_spc(z_score: float) -> bool:
    """Z-Score 유효성 검증"""
    # Z-Score가 ±10σ 초과는 센서 오류 가능성
    return -10.0 <= z_score <= 10.0
```

### 설계안 (향후 구현 시 참고)

#### 수식

```
ISC 확률 = f(전압 강하율, 온도 상승률, 전류 이상)

ISC_Score = w1 × (ΔV/Δt) + w2 × (ΔT/Δt) + w3 × I_anomaly

where:
  ΔV/Δt = 전압 강하 속도 (V/min)
  ΔT/Δt = 온도 상승 속도 (°C/min)
  I_anomaly = 전류 이상 지표
  w1, w2, w3 = 가중치 (0.4, 0.4, 0.2)
```

#### 임계값

| 등급 | ISC Score | 해석 | 조치 |
|------|-----------|------|------|
| **NORMAL** | < 0.3 | 정상 | 모니터링 |
| **WARNING** | 0.3 ~ 0.6 | 의심 | 정밀 점검 |
| **CRITICAL** | > 0.6 | 고위험 | 즉시 격리 |

#### Python 구현 (향후)

**구현 예정 위치**: `scripts/phase2_validate_safety.py` 또는 별도 모듈

```python
# 향후 구현 예정
# config/global/formulas/isc_detection.py
import polars as pl

# 가중치 상수
W_VOLTAGE = 0.4
W_TEMP = 0.4
W_CURRENT = 0.2

def calculate_isc_detection(df: pl.DataFrame) -> pl.DataFrame:
    """내부 단락 감지

    Args:
        df: 배터리 표준 파일
           필수 컬럼: timestamp, rack_id, v_avg, t_max, current

    Returns:
        컬럼 추가된 DataFrame
        - dv_dt: 전압 변화율 (V/min)
        - dt_dt: 온도 변화율 (°C/min)
        - i_anomaly: 전류 이상 지표
        - isc_score: ISC 점수 (0~1)
        - isc_risk: NORMAL/WARNING/CRITICAL
    """

    result = df.sort(['rack_id', 'timestamp']).with_columns([
        # 1. 전압 변화율
        (
            pl.col('v_avg').diff().over('rack_id') /
            (pl.col('timestamp').diff().over('rack_id').dt.total_seconds() / 60.0)
        ).alias('dv_dt'),

        # 2. 온도 변화율
        (
            pl.col('t_max').diff().over('rack_id') /
            (pl.col('timestamp').diff().over('rack_id').dt.total_seconds() / 60.0)
        ).alias('dt_dt'),

        # 3. 전류 이상 (급격한 변화)
        (
            pl.col('current').diff().over('rack_id').abs() /
            pl.col('current').std().over('rack_id')
        ).alias('i_anomaly'),
    ]).with_columns([
        # 4. ISC Score 계산 (정규화)
        (
            W_VOLTAGE * (pl.col('dv_dt').abs() / 0.1).clip(0, 1) +
            W_TEMP * (pl.col('dt_dt') / 5.0).clip(0, 1) +
            W_CURRENT * (pl.col('i_anomaly') / 3.0).clip(0, 1)
        ).alias('isc_score')
    ]).with_columns([
        # 5. 위험도 분류
        pl.when(pl.col('isc_score') > 0.6)
          .then(pl.lit('CRITICAL'))
          .when(pl.col('isc_score') > 0.3)
          .then(pl.lit('WARNING'))
          .otherwise(pl.lit('NORMAL'))
          .alias('isc_risk')
    ])

    return result
```

## 🔍 Algorithm 4: 데이터 무결성 (Integrity Check)

### 검증 규칙

```python
# config/global/formulas/integrity_check.py
import polars as pl

def check_data_integrity(df: pl.DataFrame) -> pl.DataFrame:
    """데이터 무결성 검증

    물리 법칙 위반 사항:
    1. v_min > v_max (전압 순서 위반)
    2. t_min > t_max (온도 순서 위반)
    3. v_avg < 2.0V or > 4.0V (LFP 범위 초과)
    4. t_max > 100°C (물리적 불가능)
    5. soc < 0% or > 100% (범위 초과)

    Returns:
        컬럼 추가된 DataFrame
        - integrity_ok: True/False
        - integrity_error: 오류 메시지
    """

    result = df.with_columns([
        # 물리 법칙 검증
        (
            (pl.col('v_min') <= pl.col('v_max')) &
            (pl.col('t_min') <= pl.col('t_max')) &
            (pl.col('v_avg') >= 2.0) &
            (pl.col('v_avg') <= 4.0) &
            (pl.col('t_max') <= 100.0) &
            (pl.col('soc') >= 0.0) &
            (pl.col('soc') <= 100.0)
        ).alias('integrity_ok')
    ]).with_columns([
        # 오류 메시지 생성
        pl.when(~pl.col('integrity_ok'))
          .then(
              pl.when(pl.col('v_min') > pl.col('v_max'))
                .then(pl.lit('VOLTAGE_ORDER_VIOLATION'))
              .when(pl.col('t_min') > pl.col('t_max'))
                .then(pl.lit('TEMP_ORDER_VIOLATION'))
              .when(pl.col('v_avg') < 2.0)
                .then(pl.lit('VOLTAGE_TOO_LOW'))
              .when(pl.col('v_avg') > 4.0)
                .then(pl.lit('VOLTAGE_TOO_HIGH'))
              .when(pl.col('t_max') > 100.0)
                .then(pl.lit('TEMP_TOO_HIGH'))
              .otherwise(pl.lit('SOC_OUT_OF_RANGE'))
          )
          .otherwise(pl.lit('OK'))
          .alias('integrity_error')
    ])

    return result
```

## 📊 종합 위험도 계산

### 최종 위험도 판정

```python
# config/global/formulas/risk_aggregation.py
import polars as pl

def aggregate_risk_level(df: pl.DataFrame) -> pl.DataFrame:
    """4대 알고리즘 결과 종합

    우선순위:
    1. Integrity 실패 → SENSOR_FAULT
    2. 하나라도 CRITICAL → CRITICAL
    3. 하나라도 WARNING → WARNING
    4. 모두 NORMAL → NORMAL

    Returns:
        컬럼 추가된 DataFrame
        - final_risk: 최종 위험도
        - risk_reasons: 위험 원인 목록
    """

    result = df.with_columns([
        # 최종 위험도 판정
        pl.when(~pl.col('integrity_ok'))
          .then(pl.lit('SENSOR_FAULT'))
        .when(
            (pl.col('thermal_risk') == 'CRITICAL') |
            (pl.col('spc_risk') == 'CRITICAL') |
            (pl.col('isc_risk') == 'CRITICAL')
        )
          .then(pl.lit('CRITICAL'))
        .when(
            (pl.col('thermal_risk') == 'WARNING') |
            (pl.col('spc_risk') == 'WARNING') |
            (pl.col('isc_risk') == 'WARNING')
        )
          .then(pl.lit('WARNING'))
        .otherwise(pl.lit('NORMAL'))
        .alias('final_risk')
    ]).with_columns([
        # 위험 원인 목록
        pl.concat_str([
            pl.when(pl.col('thermal_risk') != 'NORMAL')
              .then(pl.lit('THERMAL:') + pl.col('thermal_risk'))
              .otherwise(pl.lit('')),
            pl.when(pl.col('spc_risk') != 'NORMAL')
              .then(pl.lit('SPC:') + pl.col('spc_risk'))
              .otherwise(pl.lit('')),
            pl.when(pl.col('isc_risk') != 'NORMAL')
              .then(pl.lit('ISC:') + pl.col('isc_risk'))
              .otherwise(pl.lit('')),
        ], separator=', ').alias('risk_reasons')
    ])

    return result
```

## 🔄 실행 순서

```python
# scripts/run_analysis.py
from config.global.formulas import (
    thermal_safety,
    spc_balance,
    isc_detection,
    integrity_check,
    risk_aggregation
)
import polars as pl

def run_full_analysis(parquet_path: str) -> pl.DataFrame:
    """전체 분석 실행

    Args:
        parquet_path: 배터리 표준 파일 경로

    Returns:
        모든 분석 결과가 포함된 DataFrame
    """

    # 1. 배터리 표준 파일 읽기
    df = pl.read_parquet(parquet_path)

    # 2. Algorithm 1: 열적 안전성
    df = thermal_safety.calculate_thermal_safety(df)

    # 3. Algorithm 2: 셀 밸런싱
    df = spc_balance.calculate_spc_balance(df)

    # 4. Algorithm 3: 내부 단락
    df = isc_detection.calculate_isc_detection(df)

    # 5. Algorithm 4: 데이터 무결성
    df = integrity_check.check_data_integrity(df)

    # 6. 종합 위험도 계산
    df = risk_aggregation.aggregate_risk_level(df)

    return df
```

### ResultManager 사용 예시

```python
# Phase 2에서 사용하는 방법
from modules.storage.result_manager import ResultManager
import polars as pl
import json
import tempfile
import os

def save_phase2_analysis(df: pl.DataFrame, run_id: str = None):
    """Phase 2 분석 결과 저장 (ResultManager 사용)

    Args:
        df: 분석이 완료된 DataFrame
        run_id: Phase 1에서 받은 run_id (없으면 새로 생성)

    Returns:
        저장된 파일 정보
    """

    # 1. ResultManager 초기화
    manager = ResultManager(run_id)

    # 2. 임시 파일 생성
    temp_dir = tempfile.mkdtemp()

    analysis_result_path = os.path.join(temp_dir, "analysis_result.parquet")
    critical_path = os.path.join(temp_dir, "critical_racks.parquet")
    warning_path = os.path.join(temp_dir, "warning_racks.parquet")
    summary_path = os.path.join(temp_dir, "summary_stats.json")

    # 3. 임시 파일에 데이터 쓰기
    df.write_parquet(analysis_result_path)

    critical_df = df.filter(pl.col('final_risk_level') == 'CRITICAL')
    critical_df.write_parquet(critical_path)

    warning_df = df.filter(pl.col('final_risk_level') == 'WARNING')
    warning_df.write_parquet(warning_path)

    stats = {
        'total': len(df),
        'critical': len(critical_df),
        'warning': len(warning_df),
        'normal': len(df.filter(pl.col('final_risk_level') == 'NORMAL'))
    }

    with open(summary_path, 'w', encoding='utf-8') as f:
        json.dump(stats, f, ensure_ascii=False, indent=2)

    # 4. ResultManager로 Phase 2 결과 저장
    # → 자동으로 results/{run_id}/phase2_validate/로 복사하고 manifest.yaml 업데이트
    manager.save_phase2_result(
        analysis_result_path=analysis_result_path,
        critical_path=critical_path,
        warning_path=warning_path,
        summary_path=summary_path,
        duration_sec=25.3,
        stats=stats
    )

    # 5. 임시 파일 정리
    import shutil
    shutil.rmtree(temp_dir)

    return {
        'status': 'success',
        'run_id': manager.run_id,
        'phase2_dir': f'results/{manager.run_id}/phase2',
        'manifest': f'results/{manager.run_id}/manifest.yaml'
    }
```
