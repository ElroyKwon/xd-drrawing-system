## 개요

GEM(Generative Engineering Module) v2.0은 제주 BESS 배터리 시스템(160개 Rack, 3.4억 건 데이터)의 안전 진단을 자동화하는 Dual Engine 아키텍처 기반 플랫폼입니다.

### 핵심 컨셉

```
┌─────────────────────────────────────────────────────────┐
│                    GEM v2.0 Architecture                 │
│                                                           │
│  🧠 Brain (AI)          +          💪 Muscle (Python)   │
│  - Ollama/Qwen2.5                  - Pandas/Polars       │
│  - 자연어 해석                      - 병렬 연산          │
│  - 리포트 생성                      - 대용량 처리        │
└─────────────────────────────────────────────────────────┘
```

### 주요 특징

- **No-DB 전략**: 메모리 + Parquet 파일 시스템 기반
- **Multi-Site 지원**: 사이트별 배터리 맵 자동 관리
- **4대 핵심 분석**: Thermal Safety, SPC Balance, Data Integrity, System Correlation
- **자동화된 리포트**: AI 기반 전문가 리포트 생성

## Phase별 상세 흐름

### Phase 1: 데이터 변환 (Data Conversion)

```
입력: GEM/data/source/api/YYYY-MM-DD/*.csv
      또는 GEM/data/source/upload/YYYYMMDD_HHMMSS/original.csv

┌─────────────────────────────────────────────────────────────┐
│  Step 1: CSV 파일 로드                                       │
│  ┌───────────────────────────────────────────────────────┐  │
│  │  for channel in [BSC_CH_01 ~ BSC_CH_06]:             │  │
│  │      df = pd.read_csv(f"{channel}.csv")              │  │
│  │      df['channel'] = channel                         │  │
│  │      all_data.append(df)                             │  │
│  └───────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  Step 2: 데이터 검증 (Pydantic)                             │
│  ┌───────────────────────────────────────────────────────┐  │
│  │  - 시간 컬럼 존재 확인 (time, timestamp)              │  │
│  │  - 날짜 형식 검증 (YYYY-MM-DD HH:MM:SS)               │  │
│  │  - 필수 태그 존재 확인                                │  │
│  │  - 센서 값 범위 검증 (PhysicsValidator)               │  │
│  │    * 온도: -40°C ~ 85°C                               │  │
│  │    * 전압: 2.5V ~ 4.2V                                │  │
│  │    * 전류: -200A ~ 200A                               │  │
│  └───────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  Step 3: 시간 동기화 (Resampling)                           │
│  ┌───────────────────────────────────────────────────────┐  │
│  │  df = df.set_index('time')                           │  │
│  │  df_resampled = df.resample('10S').mean()           │  │
│  │  # 10초 간격으로 리샘플링                             │  │
│  └───────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│  Step 4: Parquet 변환 및 저장                               │
│  ┌───────────────────────────────────────────────────────┐  │
│  │  df.to_parquet(                                      │  │
│  │      'standard_data.parquet',                        │  │
│  │      compression='zstd',                             │  │
│  │      compression_level=9                             │  │
│  │  )                                                   │  │
│  │  # 압축률: ~70% (5GB → 1.5GB)                        │  │
│  └───────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘

출력: results/{RUN_ID}/phase1_convert/
      ├── standard_data.parquet      ← 표준화된 데이터
      └── conversion_log.json        ← 변환 로그
```

### Phase 2: 수학적 검증 (Mathematical Validation)

```
입력: results/{RUN_ID}/phase1_convert/standard_data.parquet

┌─────────────────────────────────────────────────────────────┐
│  병렬 처리 (ProcessPoolExecutor, max_workers=4)             │
│                                                              │
│  ┌──────────────────┐  ┌──────────────────┐                │
│  │  Worker 1        │  │  Worker 2        │                │
│  │                  │  │                  │                │
│  │  Thermal Safety  │  │  SPC Balance     │                │
│  └──────────────────┘  └──────────────────┘                │
│           │                      │                          │
│           ▼                      ▼                          │
│  ┌──────────────────┐  ┌──────────────────┐                │
│  │  Worker 3        │  │  Worker 4        │                │
│  │                  │  │                  │                │
│  │  Data Integrity  │  │  Correlation     │                │
│  └──────────────────┘  └──────────────────┘                │
└─────────────────────────────────────────────────────────────┘
           │                      │
           └──────────┬───────────┘
                      ▼

┌─────────────────────────────────────────────────────────────┐
│  분석 1: Thermal Safety (열적 안전성)                        │
│  ┌───────────────────────────────────────────────────────┐  │
│  │  수식: dT/dt = (T[t] - T[t-1]) / Δt                  │  │
│  │                                                       │  │
│  │  for rack in all_racks:                              │  │
│  │      temp_series = df[f'{rack}.MOD_AVG_TEMP']       │  │
│  │      dT_dt = temp_series.diff() / 10  # 10초 간격   │  │
│  │      critical = dT_dt[dT_dt > 2.0]   # 임계값       │  │
│  │      if len(critical) > 0:                          │  │
│  │          report_anomaly(rack, critical)             │  │
│  └───────────────────────────────────────────────────────┘  │
│                                                              │
│  출력: thermal_safety.json                                  │
│    {                                                        │
│      "critical": [...],  # Critical 등급 (즉시 조치)        │
│      "warning": [...],   # Warning 등급 (주의 관찰)         │
│      "normal": [...]     # 정상                            │
│    }                                                        │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│  분석 2: SPC Balance (셀 밸런싱)                             │
│  ┌───────────────────────────────────────────────────────┐  │
│  │  수식: Z-Score = (X - μ) / σ                         │  │
│  │                                                       │  │
│  │  for timepoint in all_timepoints:                    │  │
│  │      voltages = [rack.voltage for rack in racks]    │  │
│  │      mean = np.mean(voltages)                       │  │
│  │      std = np.std(voltages)                         │  │
│  │      z_scores = (voltages - mean) / std             │  │
│  │      outliers = racks[abs(z_scores) > 3.0]          │  │
│  │      if len(outliers) > 0:                          │  │
│  │          report_imbalance(outliers)                 │  │
│  └───────────────────────────────────────────────────────┘  │
│                                                              │
│  출력: spc_balance.json                                     │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│  분석 3: Data Integrity (센서 무결성)                        │
│  ┌───────────────────────────────────────────────────────┐  │
│  │  조건: Rolling Variance = 0 (30분간)                 │  │
│  │                                                       │  │
│  │  for tag in all_tags:                                │  │
│  │      rolling_var = df[tag].rolling(                 │  │
│  │          window=180  # 30분 = 180개 (10초 간격)      │  │
│  │      ).var()                                         │  │
│  │      frozen = rolling_var[rolling_var == 0]         │  │
│  │      if len(frozen) > 0:                            │  │
│  │          report_sensor_freeze(tag, frozen)          │  │
│  └───────────────────────────────────────────────────────┘  │
│                                                              │
│  출력: data_integrity.json                                  │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│  분석 4: System Correlation (시스템 상관성)                  │
│  ┌───────────────────────────────────────────────────────┐  │
│  │  분석: DC Bus 전압 vs Rack 전압 추종성                │  │
│  │                                                       │  │
│  │  dc_bus_voltage = df['DC_BUS_VOLTAGE']              │  │
│  │  for rack in all_racks:                              │  │
│  │      rack_voltage = df[f'{rack}.VOLTAGE']           │  │
│  │      correlation = np.corrcoef(                     │  │
│  │          dc_bus_voltage,                            │  │
│  │          rack_voltage                               │  │
│  │      )[0, 1]                                         │  │
│  │      if correlation < 0.8:  # 임계값                │  │
│  │          report_poor_tracking(rack, correlation)    │  │
│  └───────────────────────────────────────────────────────┘  │
│                                                              │
│  출력: system_correlation.json                              │
└─────────────────────────────────────────────────────────────┘

출력: results/{RUN_ID}/phase2_validate/
      ├── thermal_safety.json
      ├── spc_balance.json
      ├── data_integrity.json
      ├── system_correlation.json
      └── validation_log.json
```

def generate_insights_with_retry(phase2_data, max_retries=3):
    """AI 인사이트 생성 (재시도 포함)"""

    for attempt in range(1, max_retries + 1):
        try:
            # Ollama 서버 상태 확인
            if not check_ollama_health():
                raise ConnectionError("Ollama 서버 연결 실패")

            # 인사이트 생성 시도
            insights = call_ollama_api(phase2_data)

            # 성공
            log.info(f"✓ AI 인사이트 생성 성공 (시도 {attempt}/{max_retries})")
            return insights

        except Exception as e:
            log.error(f"✗ AI 인사이트 생성 실패 (시도 {attempt}/{max_retries}): {e}")

            if attempt < max_retries:
                # 재시도 대기
                wait_time = 2 ** attempt  # Exponential backoff: 2, 4, 8초
                log.info(f"  {wait_time}초 후 재시도...")
                time.sleep(wait_time)
            else:
                # 최종 실패 - 사용자 액션 필요
                show_error_dialog({
                    'error': str(e),
                    'phase2_path': f'results/{RUN_ID}/phase2_validate/',
                    'suggestions': [
                        '1. ollama serve 명령으로 서버 시작',
                        '2. ollama list로 모델 확인',
                        '3. 네트워크 연결 확인'
                    ]
                })
                raise RuntimeError("AI 인사이트 생성 최종 실패")


def check_ollama_health():
    """Ollama 서버 헬스 체크"""
    try:
        response = requests.get(
            "http://localhost:11434/api/tags",
            timeout=5
        )
        return response.status_code == 200
    except:
        return False
