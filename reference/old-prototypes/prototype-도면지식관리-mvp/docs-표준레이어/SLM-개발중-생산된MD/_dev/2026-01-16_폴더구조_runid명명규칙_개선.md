## 배경

기존 시스템의 문제점:
1. **run_id 형식 불명확**: `20260116_103045` → 무엇을 기준으로 만들었는지 알 수 없음
2. **results 폴더 위치**: 최상위에 있어서 중간 데이터와 최종 결과 구분 어려움
3. **Parquet 파일 개수**: 6개 채널 → 6개 파일 (분석 시 복잡도 증가)
4. **Phase 2 오류**: API 수집 데이터는 run_id가 없어서 검증 실패

### 1. run_id 명명 규칙 통일 (CSV 파일명 규칙과 일치)

#### CSV 파일명 규칙 (STORAGE_POLICY.md)
```
{channel}_{date_compact}_{agg_time}_{mode}_v{version}.csv
                                      ^^^^
                                      batch 또는 realtime
```

#### 새로운 run_id 형식

**Batch 모드 (과거 데이터 수집)**
```
batch_{date}_{agg_time}_{profile}_{timestamp}
예: batch_20260112_20s_NMC_LG_103045
```

**Realtime 모드 (실시간 데이터 수집)**
```
realtime_{date}_{agg_time}_{profile}_{timestamp}
예: realtime_20260116_10s_NMC_LG_103045
```

**CSV 업로드**
```
upload_{date}_{timestamp}_{profile}
예: upload_20260116_103045_NMC_LG
```

#### 장점
- ✅ 데이터 소스 명확 (batch/realtime/upload)
- ✅ 수집/분석 날짜 명확
- ✅ 집계 간격 표시 (API 수집의 경우)
- ✅ 배터리 프로파일 표시
- ✅ 같은 날짜 재분석 가능 (timestamp로 구분)

## 구현 상세

### generate_run_id() 함수
**파일**: [modules/storage/result_manager.py:15-55](../GEM/modules/storage/result_manager.py#L15-L55)

```python
def generate_run_id(mode: str, date: str = None, agg_time: str = None, profile: str = None) -> str:
    """
    run_id 생성 헬퍼 함수

    Args:
        mode: 'batch' | 'realtime' | 'upload'
        date: 데이터 날짜 (YYYY-MM-DD 또는 YYYYMMDD)
        agg_time: 집계 간격 (예: '20s', '10m')
        profile: 배터리 프로파일 (예: 'NMC_LG')

    Returns:
        run_id 문자열

    Examples:
        >>> generate_run_id('batch', '2026-01-12', '20s', 'NMC_LG')
        'batch_20260112_20s_NMC_LG_103045'

        >>> generate_run_id('upload', profile='NMC_LG')
        'upload_20260116_103045_NMC_LG'
    """
    timestamp = datetime.now().strftime("%H%M%S")

    if mode in ['batch', 'realtime']:
        # API 수집 모드
        if not date or not agg_time or not profile:
            raise ValueError(f"{mode} 모드는 date, agg_time, profile이 필요합니다")

        # 날짜 형식 통일 (YYYY-MM-DD → YYYYMMDD)
        date_compact = date.replace('-', '')
        return f"{mode}_{date_compact}_{agg_time}_{profile}_{timestamp}"

    elif mode == 'upload':
        # CSV 업로드 모드
        date_compact = datetime.now().strftime("%Y%m%d")
        if profile:
            return f"upload_{date_compact}_{timestamp}_{profile}"
        else:
            return f"upload_{date_compact}_{timestamp}"

    else:
        raise ValueError(f"Unknown mode: {mode}")
```
