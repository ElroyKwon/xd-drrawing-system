## 🔑 핵심 기능

### 1. API 데이터 수집
- **과거 데이터**: 날짜별 배치 수집 → `data/source/api/YYYY-MM-DD/`
- **실시간 데이터**: 5분 간격 자동 수집
- **채널별 선택**: BSC_CH_01~06 중 선택 가능

### 2. CSV 업로드
- 웹 UI에서 직접 업로드
- `data/source/upload/YYYYMMDD_HHMMSS/` 에 저장

### 3. 배터리 안전 분석
- **Phase 1**: 데이터 변환 (CSV → Parquet)
- **Phase 2**: 4대 안전성 검증
  - 열적 안전성 (Thermal Safety)
  - 셀 밸런싱 (SPC Balance)
  - 내부 단락 (Internal Short Circuit)
  - 데이터 무결성 (Data Integrity)
- **Phase 3**: AI 인사이트 생성 (Ollama)
- **Phase 4**: HTML 리포트 생성

### 4. REG 학습 데이터 생성 (🆕)
- 분석 결과 → 학습 데이터 자동 변환
- `data/training/processed/` 에 저장
- 상세: [05_REG_학습_가이드.md](05_REG_학습_가이드.md)
