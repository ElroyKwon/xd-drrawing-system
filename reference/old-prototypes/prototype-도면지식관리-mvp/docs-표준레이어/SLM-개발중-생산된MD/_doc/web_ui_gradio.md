# GEM 웹 UI 사용 가이드 ## 🚀 빠른 시작 ### 1. 의존성 설치 ```bash # Gradio 설치 (필수) pip install gradio==4.13.0 # 또는 전체 의존성 설치 pip install -r requirements.txt ``` ### 2. 웹 UI 실행 ```bash # GEM 디렉토리에서 실행 python app.py ``` 서버가 시작되면 자동으로 브라우저가 열립니다: - **로컬 URL**: http://localhost:7860 - **네트워크 URL**: http://0.0.0.0:7860 --- ## 📋 사용 방법 ### Tab 1: 📂 Phase 1 - 데이터 변환 **CSV → Parquet 변환 + 배터리 프로파일 자동 감지** 1. **CSV 파일 업로드** - XD-HUB CSV 파일 선택 - 드래그 앤 드롭 또는 파일 선택 버튼 클릭 2. **배터리 프로파일 선택** - `auto (자동 감지)`: 전압 범위 기반 자동 감지 (권장) - `NMC_LG`: 주력 배터리 (1.5°C/min) - `NCMA_LG`: 차세대 고니켈 배터리 (1.2°C/min) - `LFP_LG`: 안전성 우선 배터리 (2.0°C/min) - `GENERIC_ESS_SAFE`: 범용 안전 프로파일 (1.0°C/min, 임시 사용만) 3. **▶️ Phase 1 실행** 버튼 클릭 4. **결과 확인** - 📊 실행 상태: 성공/실패 메시지 - 📋 실행 로그: 6단계 진행 상황 - 🔍 프로파일 정보: 감지된 프로파일, 신뢰도, 파일 정보 **Phase 1 실행 단계:** ``` [1/6] 파일 확인 [2/6] 배터리 프로파일 처리 [3/6] CSV 데이터 로딩 [4/6] 데이터 변환 [5/6] Parquet 저장 [6/6] ResultManager 등록 ``` --- ### Tab 2: 📊 Phase 2 - 안전성 분석 **배터리 안전성 알고리즘 분석 (미구현)** ⚠️ **현재 상태**: Phase 2는 아직 구현되지 않았습니다. **구현 예정 알고리즘:** 1. **Thermal Safety**: dT/dt 열변화 감지 2. **SPC Balance**: Z-Score 셀 밸런싱 분석 3. **ISC Detection**: 내부 단락 감지 4. **Integrity Check**: 센서 오류 검증 **사용 방법 (구현 후):** 1. Phase 1 완료 후 자동으로 활성화 2. **▶️ Phase 2 실행** 버튼 클릭 3. 분석 로그 및 결과 확인 --- ### Tab 3: 📄 Phase 3 - 리포트 생성 **Ollama 인사이트 생성 및 HTML 리포트 (미구현)** ⚠️ **현재 상태**: Phase 3는 아직 구현되지 않았습니다. **구현 예정 기능:** - Ollama (qwen2.5:32b) 연동 - AI 기반 인사이트 생성 - HTML 최종 리포트 생성 - 다운로드 기능 **사용 방법 (구현 후):** 1. Phase 2 완료 후 자동으로 활성화 2. **▶️ Phase 3 실행** 버튼 클릭 3. HTML 리포트 다운로드 --- ### Tab 4: 💬 AI 채팅 **Ollama와 대화하기 (미구현)** ⚠️ **현재 상태**: Ollama 연동은 아직 구현되지 않았습니다. **구현 예정 기능:** - 배터리 분석 결과 해석 - 기술 문서 참조 답변 - 알고리즘 설명 - 실시간 질의응답 **사용 방법 (구현 후):** 1. 하단 입력창에 질문 입력 2. **전송** 버튼 클릭 또는 Enter 3. AI 응답 확인 --- ## 🔋 배터리 프로파일 가이드 ### 자동 감지 (auto) 전압 범위를 기반으로 자동 감지: | 프로파일 | 전압 범위 (V) | 공칭 전압 (V) | 제조사 | |---------|-------------|-------------|--------| | NMC_LG | 3.5 ~ 4.15 | 3.7 | LG Energy Solution | | NCMA_LG | 3.6 ~ 4.2 | 3.7 | LG Energy Solution | | LFP_LG | 2.8 ~ 3.6 | 3.2 | LG Energy Solution | **신뢰도 기준:** - ≥80%: 자동 적용 (AUTO) - 50~79%: 사용자 확인 필요 (CONFIRM) - <50%: 수동 선택 필요 (MANUAL) ### 수동 선택 프로파일을 직접 선택할 수 있습니다: #### NMC_LG (권장 - 주력) - **용도**: 대규모 ESS, ESS 시스템 - **특징**: 고에너지밀도, 검증된 NMC 기술 - **임계값**: dT/dt 1.5°C/min, Z-Score 2.5σ - **수명**: 3500 사이클, 10년 #### NCMA_LG (차세대 고성능) - **용도**: 차세대 대규모 ESS, 프리미엄 VPP - **특징**: 90% 니켈, 최고 에너지밀도 - **임계값**: dT/dt 1.2°C/min (가장 엄격), Z-Score 2.0σ - **수명**: 4000 사이클, 12년 - ⚠️ **주의**: 열관리 시스템 필수 #### LFP_LG (안전성 최우선) - **용도**: 장수명 ESS, 안전성 중시 프로젝트 - **특징**: 최고 안전성, 최장 수명 - **임계값**: dT/dt 2.0°C/min (여유), Z-Score 3.0σ - **수명**: 8000 사이클, 20년 - ✅ **장점**: 열폭주 위험 거의 없음 (270°C) #### GENERIC_ESS_SAFE (임시 사용만) - **용도**: 배터리 타입 불명 시 임시 사용 - **특징**: 매우 보수적 임계값 - **임계값**: dT/dt 1.0°C/min (가장 엄격), Z-Score 2.5σ - ⚠️ **경고**: - 오탐(False Positive) 가능성 높음 - 임시 사용만 권장 - 가능한 빨리 정확한 프로파일로 변경 --- ## 📁 출력 파일 구조 ``` GEM/ ├── data/ │ └── standard/ │ └── 2025/ │ └── 12/ │ └── battery_data_20251212_143000.parquet # Phase 1 출력 │ └── results/ └── 20251212_143000/ # Run ID ├── manifest.yaml # 메타데이터 ├── phase1/ │ └── battery_data_20251212_143000.parquet ├── phase2/ # 미구현 │ └── analysis_results.parquet ├── phase3/ # 미구현 │ └── insights.json └── final/ # 미구현 └── report.html ``` --- ## 🐛 문제 해결 ### Gradio 설치 오류 ```bash # pip 업그레이드 python -m pip install --upgrade pip # Gradio 재설치 pip uninstall gradio pip install gradio==4.13.0 ``` ### "Module not found" 오류 ```bash # GEM 디렉토리에서 실행하는지 확인 cd D:\0001_project\RFS\GEM python app.py ``` ### "Config not found" 오류 ```bash # config 디렉토리 확인 ls config/central_config.yaml ls config/battery_profiles/ # 없으면 프로젝트 구조 확인 ``` ### Phase 1 실행 오류 1. **CSV 파일 형식 확인** - XD-HUB 형식인지 확인 - 필수 컬럼: time, rack_name, module_name, voltage_avg, temp_avg, current, soc 2. **Polars 설치 확인** ```bash pip install polars==0.20.3 ``` 3. **디스크 공간 확인** - Parquet 파일 저장을 위한 충분한 공간 필요 --- ## 🔧 고급 설정 ### 포트 변경 `app.py` 수정: ```python demo.launch( server_name="0.0.0.0", server_port=8080, # 원하는 포트로 변경 share=False, show_error=True ) ``` ### 외부 접속 허용 ```python demo.launch( server_name="0.0.0.0", # 모든 IP 허용 server_port=7860, share=True, # Gradio 공개 링크 생성 show_error=True ) ``` ⚠️ **보안 주의**: 프로덕션 환경에서는 방화벽 및 인증 설정 필수 --- ## 📚 추가 문서 -
