## 1. 이미지 캡쳐 현황

**브라우저 자동화 도구(Playwright/Puppeteer)가 환경에 없어 스크린샷 자동 캡쳐는 불가능합니다.**  
대신 HTTP SSR 응답을 직접 파싱해서 렌더링 결과를 검증했습니다.  
이미지 캡쳐는 브라우저에서 직접 F12 → Network 탭 또는 수동 캡쳐 필요.
