## 팀 공유 패키지

팀원과 다른 컴퓨터에서 재사용할 패키지는 아래에 둔다.

```text
05-팀공유-스킬패키지/
```

핵심 구성:

- 설치 가능한 `SKILL.md` 후보
- 스킬별 `evals/evals.json`
- Windows 설치/검증 스크립트
- 프로젝트 루프 템플릿
- 설치/테스트/레지스트리 문서

현재 상태는 `검증 가능한 배포 후보`이며, 팀 표준 확정 전에는 빈 테스트 프로젝트에서 검증해야 한다.
