# AI 개발 루프 스킬 학습

## 목적

이 폴더는 [[인공지능]] 개발 루프, [[Claude Code]], [[Codex]], [[Warp]] 터미널 기반 학습 방식을 따라 하기 위해 만든 전용 정리 공간이다.

현재 단계는 사용자가 제공하는 3개 텍스트 자료와 영상 링크를 모두 수집한 뒤, 원문 주장과 실제 영상 정보, 설명란, 고정 댓글, 자막/챕터를 대조해 학습용 문서와 실행 루프를 만드는 준비 단계이다.

## 현재 상태

- 자료 수집: 3/3 수신
- 영상 직접 검증: 1차 진행
- 설명란 확인: 1차 진행
- 고정 댓글 확인: 대기
- 스킬 다운로드/설치 판단: 대기
- `CLAUDE.md`, 글로벌 `CLAUDE.md`, `AGENTS.md` 반영: 설계 완료 / 실제 반영 대기
- 중요 세션 인수인계: `_NEXT-SESSION.md` 생성 완료

## 폴더 구조

| 폴더 | 용도 |
|---|---|
| `00-원문-입력/` | 사용자가 붙여 넣은 텍스트와 영상 링크의 인입 정리 |
| `01-분석노트/` | 직접 검증 전후의 분석 노트 |
| `02-스킬-후보/` | 다운로드하거나 직접 만들 스킬 후보 정리 |
| `03-에이전트-규칙-반영/` | `CLAUDE.md`, 글로벌 `CLAUDE.md`, `AGENTS.md` 반영 후보 |
| `04-실습-루프/` | 실제 따라 할 루프, 체크리스트, 실행 템플릿 |
| `05-팀공유-스킬패키지/` | 팀원과 공유할 설치 가능 스킬 후보, 템플릿, 설치/검증 가이드 |

## 작업 원칙

1. 사용자 제공 텍스트는 근거 자료로 보존하되, 사실로 확정하지 않는다.
2. 유튜브 영상 본문, 설명란, 고정 댓글, 챕터/자막을 직접 확인한 뒤 최종 정리한다.
3. 다른 [[대규모언어모델]]이 만든 요약은 누락·왜곡 가능성이 있으므로 원본과 대조한다.
4. 스킬 설치나 글로벌 규칙 수정은 후보 문서를 만든 뒤 영향 범위를 확인하고 진행한다.
5. 최종 목표는 단순 요약이 아니라, 사용자가 [[Warp]] 터미널에서 반복적으로 사용할 수 있는 개발 루프와 스킬 체계를 만드는 것이다.

## 다음 단계

3개 입력 자료 수집은 완료했다. 1차 공식 문서 확인 결과, 첫 실습은 API 키 없이 계정 로그인으로 가능한 `[[Warp]] 4페인 + [[Claude Code]] + [[Codex CLI]]/GPT + [[Gemini CLI]] + PowerShell` 조합으로 진행한다. 자세한 기준은 `01-분석노트/05-API키-없이-계정로그인-가능범위.md`와 `01-분석노트/06-모델-역할-및-터미널-재현계획.md`를 따른다.

다음 세션에서 바로 이어갈 기준 문서는 `_NEXT-SESSION.md`다.

## 팀 공유 패키지

팀원 또는 다른 컴퓨터의 AI가 이 폴더를 읽고 사용할 때는 아래 문서부터 시작한다.

```text
05-팀공유-스킬패키지/README.md
```

포함 내용:

- `project-bootstrap`
- `planning-gate`
- `validator-loop`
- `evidence-report`
- `tag-alarm-review`
- Windows 설치 스크립트
- 스킬 패키지 검증 스크립트
- 프로젝트 루프 템플릿

주의: 이 패키지는 설치 가능한 형식이지만, 팀 표준으로 확정하기 전에는 빈 테스트 프로젝트에서 `evals` 시나리오로 검증해야 한다.
